TeachProxy Student API Key Tester

Verify a student's API key can call the OpenAI-compatible endpoint POST /v1/chat/completions. This page runs fully in your browser; the API key is only sent in the request header.

Tip: treat student API keys like passwords. If leaked, regenerate.
